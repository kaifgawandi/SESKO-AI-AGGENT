# SESKO-AI-AGGENT

SESKO is a sophisticated, full-stack AI Assistant designed to bridge the gap between local LLM processing and real-time web intelligence. Built with a focus on privacy and productivity, it combines deep research capabilities, multi-modal OCR, and a sleek, "Gemini-inspired" user experience.

🛠️ The Technical Stack (Developer's View)

Backend: FastAPI (Python) – High-performance asynchronous execution.

Brain: Llama 3.2 (via Ollama) – Local, private, and powerful.

Real-time Intel: Google Custom Search API integration.

Vision/OCR: Pytesseract (Tesseract OCR) & PDFPlumber for document analysis.

Database: SQLite3 – Lightweight, session-based chat history and user authentication.

Frontend: Pure HTML5/CSS3/JS – Responsive "Dark Mode" UI with Markdown rendering and PDF export.



✨ Key Features

Deep Research Mode: Aggregates data from multiple search queries to generate comprehensive reports.

Multi-Modal Uploads: Extract text from Images (JPG/PNG) and PDFs instantly using integrated OCR.

Rinz Mode: A creative engine for image generation and artistic prompts.

Smart Sessions: Persistent chat history, ability to pin important chats, and rename sessions.

Export Ready: Download any AI response as a professional PDF or Markdown file.
